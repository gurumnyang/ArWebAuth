## Installation

To install the dependencies and set up the project, run the following command:  
프로젝트의 의존성을 설치하고 설정하려면 다음 명령어를 실행하세요:

```bash
npm install
```

Or if you're using `yarn`:  
혹은 `yarn`을 사용하는 경우:

```bash
yarn install
```

## Usage

To start the development server, run:  
개발 서버를 시작하려면 다음 명령어를 실행하세요:

```bash
npm start
```

Or:  

```bash
yarn start
```
